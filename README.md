# E-commerce  — электронная торговля, или электронная коммерция. Ядро для интернет-магазина.

## Описание:

Приложение для создания ядра интернет-магазина.

## Возможности:

- ****Товары (Product)****
Класс хранит характеристики товара (название, описание, цена, количество)

- ****Категории (Category)****
Класс хранит характеристики категории (наименование категории, описание, список товаров)

Реализована подгрузка данных по категориями и товарам из файла JSON в модуле utils.py: 
- Функция read_json_file производит чтение файла JSON.
- Функция create_objects_from_json создавает объекты классов из файла JSON.
